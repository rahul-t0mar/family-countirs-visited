Listed below is step by step process to use this project.  
(1) Download the zip file and unzip it.    
(2) Install the postgres and the pgadmin 4    
(3) Open the extracted folder in the vscode
(4) Install the npm packages usking the command   npm i   
(5) Open the PgAmin 4 ui and create a database.  
(6) Go to the database and create the table for countries and import the data from the countries.csv file.    
(7) Then follow the database commands in the queries.sql file uploaded.    
(8) Go to the index.js page fill the name of the database created by you and the postgres password.   
(9) At last open terminal, type the command :    node index.js   or    nodemon index.js   
(10) Go to the webpage http://localhost:3000 to see the final result here you can create new users and add new country names to the list.
